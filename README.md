# Proyecto NOC
Crear una serie de tareas usando arqutectura limpia
con typescript

# dev
1. Clonar el archivo .env.template a .env
2. Configurar las variables de entorno

```
PORT = 3000

MAILER_EMAIL = 
MAILER_SECRET = 
PROD = false
```
3. Ejecutar el comando ```npm install```
4. ejecutar ``` npm run dev ```
